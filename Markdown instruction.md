## Списки

Чтобы добавить ненумерованые списки необходимо пункты выделить (*). 
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованые списки необходимо пункты просто пронумеровать. 
Например, вот так:
1. Первый пункт
2. Второй пункт
